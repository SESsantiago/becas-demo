Contexto

Becas Para Todos es una aplicación para la administración de becas escolares facilitando el proceso de admisión en las entidades que las ofrecen. La aplicación busca agilizar y orientar a los usuarios que algunas veces por desconocimiento no saben el proceso para obtener una beca. Las personas a menudo por la desinformación de no saben que becas ofrecen algunas instituciones y esto acaciana que muchas se pierdan.
